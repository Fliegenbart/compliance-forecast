  • Dieses System gibt beratende Risikosignale. Es trifft keine Entscheidungen über:
  • keine Batchfreigabe
  • keine CAPA-Genehmigung
  • keine Deviation-Schließung
  • keine finale Root-Cause-Bestätigung
  • keine Supplier Qualification
  • keine Validierungsfreigabe
  • keine regulatorische Meldeentscheidung
  • Alle Signale sind Hinweise zur Priorisierung menschlicher Prüfung — keine Compliance-Aussagen.
